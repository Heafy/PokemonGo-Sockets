### Requisitos

* [Python 3.7.+](https://www.python.org/)
* [ImageMagick](https://www.imagemagick.org/script/index.php    )
* [Pillow 2.7.0](https://pypi.org/project/Pillow/2.7.0/)

### Ejecución

#### Servidor

```bash
cd src/
python server.py
```

#### Cliente

```bash
cd src/
python client.py <IP_SERVIDOR> <PUERTO>
```
